# Security-Projects

A collection of small security-focused projects and experiments.

## Projects

- **`port_scanner.py`**  
  A multithreaded TCP port scanner (`Threader 3000`) that scans ports `1-65535`, reports open ports, and suggests an `nmap` command for deeper service/version checks.

- **`pythonkeylogger.py`**  
  A Python keylogger script built with `pynput` that captures keystrokes and batches them for email delivery through a helper `send_email` module.

- **`encryption_decryption_betounsi/`**  
  A Java console project (`keos_crypto_alg`) that provides custom string encryption/decryption using a word-mapping cipher and an interactive menu for choosing encrypt/decrypt operations.

## Note

Use these tools only in authorized, legal environments (e.g., labs, CTFs, or systems you own/have explicit permission to test).
